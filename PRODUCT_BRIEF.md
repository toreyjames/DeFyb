# PRODUCT BRIEF: DeFyb

## Positioning

DeFyb is not a generic AI scribe. It is a revenue intelligence system that converts clinical encounter evidence into compliant billing outcomes.

## Core Product Loop

1. Encounter evidence intake (transcript, note, audio context)
2. Clinical fact extraction
3. Billing recommendation with rationale
4. Documentation gap detection
5. Revenue impact projection
6. Agentic revenue-capture action queue for provider + billing + ops workflows

## MVP Scope

- No authentication
- No EHR integrations
- Deterministic, rules-based logic
- Clear UI for provider review and approval
- DME and prior authorization modules deferred until after MVP validation

## Primary Outcome

Increase compliant revenue capture by reducing undercoding and improving documentation completeness.
